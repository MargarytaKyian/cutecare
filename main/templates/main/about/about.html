Суть Проєкту

"CuteCare" – це дипломний проєкт на тему "Веб-застосунок автоматизованих ветеринарних консультацій". Його мета – надати комплексну платформу, що поєднує зручність електронної комерції зоотоварів та доступність первинних ветеринарних порад.

Ключові напрямки:

  • Електронна комерція: Широкий асортимент зоотоварів з можливістю безпечної онлайн-оплати через платіжну систему Stripe.
  • AI-Консультації: Автоматизовані ветеринарні консультації на базі штучного інтелекту Gemini, що допоможуть отримати швидкі поради.
  • Комплексний догляд: Поєднання продажу товарів та надання інформаційної підтримки для всебічної турботи про домашніх тварин.

Технології та Інструменти

Проєкт реалізовано з використанням сучасних веб-технологій для забезпечення надійності, швидкості та зручності користування.

Python & Django

Серверна частина розроблена на Python з використанням фреймворку Django, що забезпечує швидку розробку та надійну архітектуру.

Tailwind CSS

Адаптивний та сучасний інтерфейс користувача створено за допомогою Tailwind CSS для гнучкого та швидкого стилювання.

Alpine.js

Для створення легкої клієнтської інтерактивності та динамічних компонентів використовується мінімалістичний JavaScript фреймворк Alpine.js.

Stripe API

Інтеграція з Stripe API для безпечної та надійної обробки онлайн-платежів за товари.

Gemini API

Використання Gemini API для реалізації функціоналу AI-чатбота, що надає ветеринарні консультації.

Font Awesome

Широкий набір масштабованих векторних іконок надається бібліотекою Font Awesome для покращення візуального сприйняття інтерфейсу.

AOS - Animate On Scroll

Ефекти анімації при прокручуванні сторінки реалізовані за допомогою бібліотеки AOS.css, що додає динаміки елементам.

Адаптивний Дизайн

Веб-застосунок повністю адаптивний та коректно відображається на всіх пристроях: комп'ютерах, планшетах та смартфонах.

Каталог та Відгуки

Інтуїтивно зрозумілий каталог товарів та послуг, система рейтингів та відгуків для допомоги у виборі.
